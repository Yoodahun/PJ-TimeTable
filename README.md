# PJ TimeTable



<img align="right" width="500" height="300" src="app/assets/images/pjTimeTable.png">

あるタスクごとに時間を記録し、計算するWebアプリです。



## Getting Started

`rails db:migrate`

- Create Database

`rails db:seed`

- Initialize DB data

`rails s`

- Running local server
